Final test

  1. 請確認點名網頁中的學號與 Github 帳號對應資料, 學期成績評分將依據此一連結. 若有更動請透過 studlist 中的 Pull Requests 提出更改.
  2. 期末 Final Project 頁面 (包含本週與下週的考試內容), 必須以學號為標題, 在個人的作業動態網站上製作完成 H3 頁面內容後, 將超文件存為"學號_html.txt", 並以 Pull Requests 送至期末倉儲 downloads 目錄後, 由各組組長合併至 cad2021_final, 後續學員內容導入與靜態網頁轉換將透過程式統一處理, 各組組長只要負責將學員的期末報告網頁合併至 downloads 即可.
  3. 本週考試題目共有兩題, 第一題如先前公布, 請學員自選 CAD 套件, 完成零組件繪圖後, 轉入 CoppeliaSim, 以 Slider 0-360 度控制藍色搖桿轉動.
  4. 本週考試第二題為延伸第一個題目, 以相同尺寸完成對稱之連桿機構, 從 CAD 套件轉入 CoppeliaSim 後, 以 Slider 0-360 度控制 Point0-Point1 搖桿轉動.
  5. 請各學員分別以圖片及影片檔案, 呈現零件繪圖, 轉檔, 以及模擬場景製作的關鍵畫面, 並在送至期末倉儲 downloads 頁面中加上製作心得後, 請組長將各學員的期末頁面合併至 cad2021_final 倉儲中.
  6. 完成上述步驟後, 請將相關連結在本討論區回報, 並提供本學期的課程學習心得.

最後請各組組長確認一下組員學號都已經登錄到 https://github.com/mdecourse/cad2021_final/blob/main/downloads/2b_grouping.txt
